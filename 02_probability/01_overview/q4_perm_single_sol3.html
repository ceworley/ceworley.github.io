This scenario describes a permutations problem (order matters). We are considering the nonrepeating sequences of size 3 from a set of size 12.

\[\begin{aligned} {_nP_r} &= \frac{n!}{(n-r)!} \\\\ n &= 12 \\\\ r &= 3 \\\\ {_{12}P_{3}} &= \frac{12!}{(12-3)! } \\\\ &= \frac{12!}{9!} \\\\ &= 12\cdot11\cdot10 \\\\ &= 1320 \end{aligned}\]
